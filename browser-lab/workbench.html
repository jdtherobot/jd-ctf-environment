🧰 Open the toolbox?

Every challenge here can be solved with a small set of command-line tools. Part of the fun is figuring out which tool fits — so they stay out of sight until you ask.

⌨️ Terminal basics

The terminal on the right is a real Linux computer running in your browser. Your challenge files are already on it. A few commands to get moving:

cd ~/challenges     # go to the folder with your files
ls -la              # list what's there
cat BRIEF-*.md      # print a briefing to the screen
less email.eml      # scroll a big file  (press q to quit)
file *              # ask what each file really is

Ctrl+C stops a running command. Use your OS paste shortcut to paste. Stuck on which tool to use? Open Help → Reveal the toolbox.
