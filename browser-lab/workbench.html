⌨️ Terminal basics

    
The terminal on the right is a real Linux computer running in your browser. Your challenge files
    are already on it. A few commands to get moving:

    
cd ~/challenges     # go to the folder with your files
ls -la              # list what's there
cat BRIEF-*.md      # print a briefing to the screen
less email.eml      # scroll a big file  (press q to quit)
file *              # ask what each file really is

    
Ctrl+C stops a running command.
    Use your OS paste shortcut to paste. Not sure what tools you have? Open ▦ Applications.
