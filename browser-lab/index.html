The graphical desktop isn't on this host

The Desktop runs on CheerpX, which needs the browser to be cross-origin isolated — that requires two HTTP response headers:

Cross-Origin-Opener-Policy: same-origin
Cross-Origin-Embedder-Policy: require-corp

GitHub Pages can't set custom response headers, so SharedArrayBuffer (which CheerpX depends on) is unavailable here. Two supported ways to get the Desktop:

  • Deploy this folder to Cloudflare Pages with a _headers file (both headers included in the repo).
  • Or add the coi-serviceworker shim to this GitHub Pages site to synthesize isolation client-side.

Meanwhile, the Terminal gives you the complete toolkit and solves every challenge today.
