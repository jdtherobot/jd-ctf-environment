# Challenge 4 — Stegosaurus 3

A single, innocent-looking photo — `Honey.jpeg` — came through. It opens fine. It looks fine.

It is not fine.

There is more than one thing hidden in this file, and more than one layer between you and
the truth. Some of what you'll find is a trap. Some of it is the real thing wearing a disguise.
Carve it apart, crack what you can, and keep digging: the last secret is protected by the ones
before it.

**Your goal:** recover the flag. It looks like `Flag{...}`.

**What you have:** just this one file, `Honey.jpeg`.

> Tip: one file can be many files stacked together. Start by asking what's *really* inside.
